# django

 下載django
 ---------
 __Anaconda虛擬環境下載:__
 ```shell

  conda install -c anaconda django

 ```
 __pip下載:__

 ```shell
    
   py -m pip install Django

 ```
 查看django版本
 -------------
 ```shell 

   python -m django --version 

 ```
 創建django專案
 -------------
 ```shell

  django-admin startproject project

 ```
 測試django
 ---
```shell
 
 python manage.py runserver

```
瀏覽器輸入127.0.0.1:8000在本機測試django
